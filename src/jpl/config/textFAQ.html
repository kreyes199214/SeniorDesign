To best experience Moon Trek, we recommend using one of the following WebGL enabled browsers:
  1. Google Chrome 18+
  2. Mozilla Firefox 40.0+
  3. Apple Safari 8+

Older browser versions will work, but you will not have the ability to experience the 3D globe. To check if your browser supports WebGL, you can visit https://get.webgl.org.

Internet Explorer Support: Internet Explorer is not currently supported. We suggest using a modern version of Chrome, Firefox or Safari.
